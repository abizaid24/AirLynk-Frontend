# 🚀 Overview

AirLynk is a full-stack airline reservation platform that combines a premium user interface with a scalable backend architecture.

The platform enables users to search flights, manage bookings, select seats, receive price alerts, interact with an AI assistant, and securely authenticate using JWT-based authentication.

The project has been developed with scalability, clean architecture and production deployment in mind.

# 💻 Local Development

```bash
git clone https://github.com/YOUR_USERNAME/AirLynk-Frontend.git

cd AirLynk-Frontend

npm install

npm run dev
```

Production Build

```bash
npm run build

npm start
```
